# Поддержка редактирования названий наборов

Сейчас имя набора выглядит как path до него в иерархии заголовков. Нужно хранить больше информации о всех радительских наборах и о месте в файлах, где набор определяется, чтобы иметь возможность редактировать название набора через утилиту.

Стоит подумать о назначении id для наборов и добавлении parent-id вместо использования иерархии пути в строке. Тогда можно будет и в issue добавить set-id.

При выводе списков можно сделать ещё один режим, повторяющий вид markdown с отображением уровня их заголовка.
